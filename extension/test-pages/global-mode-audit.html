Missing page language

This document has no lang attribute on <html> — flagged for manual review, not auto-fixed. Global Mode used to guess a value from the browser's locale (navigator.language), but that's the visitor's locale, not necessarily the page's actual content language, so a wrong guess could make a screen reader mispronounce the entire page. Safer to leave it to a human than to silently apply a guess.
